#1 Tasks
To commit your changes in git:
1. Stage your changes:
git add .
(This stages all modified files, or you can specify individual files)
2. Commit with a message:
git commit -m "commit msg"
3. Push to remote repository (optional):
git push

#2 Tasks
 semantic tags:
<header> → intro content for the page
<nav> → navigation links only
<main> → the primary content of the page
<section> → grouped content with a theme
<article> → standalone, reusable content
<footer> → closing info, metadata, links

#3 Tasks
 Add paragraphs using <p>
 Use <strong> and <em> meaningfully
 Add:
 One unordered list <ul> <li>
 One ordered list <ol> <li>
 Add at least 5 links: <a>
 One external
 One internal placeholder
